## 三、Prompt 组装方式

**代码位置**: `agent/nodes/prompt_builder.py` 的 `build_prompt()` 函数

组装逻辑：**上下文 + 模板**

```
user_prompt =
    ## 排盘:\n{PAIPAN输出}
    ## 古籍:\n{PAIPAN古籍}
    ## 整体分析:\n{OVERALL输出}
    ## 十神:\n{SHISHEN输出}
    ## 格局判断:\n{GEJU_ROUTER输出}      # 如果当前节点不是GEJU系列
    ## 格局详细分析:\n{GEJU_ANALYSIS输出}  # 如果当前节点不是GEJU系列
    ## 格局层次:\n{GEJU_LEVEL输出}       # 如果当前节点不是GEJU系列
    ## 五行喜忌:\n{WUXING_PREFS输出}
    + {该节点的模板内容}
```

**规则**:
- 只注入当前节点**能看到**的上游输出（按依赖关系）
- 如果上游输出为空，跳过该段落
- system_prompt 固定为 `"你是一位精通八字的算命师..."`
